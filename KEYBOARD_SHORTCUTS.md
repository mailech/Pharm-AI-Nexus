# PharmAI Nexus - Keyboard Shortcuts ⌨️

## Analysis Controls
- **Ctrl + Enter** - Run analysis (when 2+ drugs are selected)
- **Escape** - Clear current analysis results

## Navigation
- **Tab** - Navigate between input fields
- **Enter** (in drug input) - Add drug to list

## Tips
- Use keyboard shortcuts during demos for a smoother presentation
- The shortcuts work globally, so you don't need to focus specific elements
